# Instagram Clone

An Instagram-inspired web application built with HTML, CSS, and JavaScript. This project recreates the core look and feel of a social media feed while serving as a practical exercise in front-end development and modern web design.

## Overview

The goal of this project is to strengthen my understanding of HTML, CSS, and JavaScript by building a familiar interface from scratch and gradually adding new functionality.

## Features

* Responsive layout
* User profile and post sections
* Clean and semantic HTML structure
* Modern CSS styling using Flexbox
* JavaScript-powered interactivity
* Organized and maintainable codebase

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Future Improvements

* Dynamic posts generated from JavaScript data
* Like button functionality
* Dark mode
* Improved accessibility
* Additional responsive enhancements


## Purpose

This repository is part of my web development portfolio and documents my progress as I continue learning front-end technologies and best practices.
